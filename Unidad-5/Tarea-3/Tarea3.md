# Trabajando con Join´s

Realiza las siguientes consultas:

- Mostrar todos los clientes.
```sql
select * from clientes;

| id_cliente | nombre_cliente | ciudad_cliente |

|          1 | Juan           | Ciudad A       |
|          2 | María          | Ciudad B       |
|          3 | Pedro          | Ciudad C       |
+------------+----------------+----------------+

```

```sql
select * from ordenes;
+----------+------------+-------------+
| id_orden | id_cliente | fecha_orden |

|        1 |          1 | 2024-03-01  |
|        2 |          2 | 2024-03-02  |
|        3 |          3 | 2024-03-03  |
```

